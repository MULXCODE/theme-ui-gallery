### Simple Button

A button will automatically be styled by the variant in the theme configuration for button.

This button: 

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button>Primary</Button>`}</LiveCode>

Is the same as this with a theme file containing this variant defined:

```
theme: {
  buttons: {
    primary: {
      color: "background",
      bg: "primary",
    },
  }
}
```

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button variant="buttons.primary">Primary</Button>`}</LiveCode>

### Using other variants

You can use other variants that are defined by referencing them explicitly:

```
theme: {
  buttons: {
    primary: {
      ...
    },
    secondary: {
      variant: "buttons.primary",
      color: "background",
      bg: "secondary",
    },
    ghost: {
      color: "text",
      bg: alpha("primary", 0.1),
      "&:hover": {
        bg: alpha("primary", 0.2),
      },
    },
  }
}
```

With the above theme configuration you can then reference them:

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button variant="buttons.secondary">Secondary</Button>
<Button variant="buttons.ghost">Ghost</Button>`}</LiveCode>

### Rounded Buttons

Using the sx prop, you can override specific styles that are being applied from the theme configuration.

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button sx={{ borderRadius: '0' }}>Primary</Button>
<Button sx={{ borderRadius: '1' }}>Primary</Button>
<Button sx={{ borderRadius: '2' }}>Primary</Button>
<Button sx={{ borderRadius: '3' }}>Primary</Button>`}</LiveCode>

You can pass in the value of an index instead to pull the value from an array in the theme configuration:

```
theme: {
  radii: [0, 2, 4, 8],
}
```

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button sx={{ borderRadius: '0' }}>Primary</Button>
<Button sx={{ borderRadius: '1' }}>Primary</Button>
<Button sx={{ borderRadius: '2' }}>Primary</Button>
<Button sx={{ borderRadius: '3' }}>Primary</Button>`}</LiveCode>

### Colored Buttons

Any styles can be 

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button sx={{ backgroundColor: 'primary' }}>Primary</Button>
<Button sx={{ backgroundColor: 'secondary' }}>Secondary</Button>`}</LiveCode>

### Icon Buttons

Theme UI includes an `IconButton` component.

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<IconButton
  aria-label='Toggle dark mode'>
  <svg sx={{ height: '14px', width: '14px'}} viewBox="0 0 20 20">
    <path d="M13 8V2H7v6H2l8 8 8-8h-5zM0 18h20v2H0v-2z"/>
  </svg>
</IconButton>`}</LiveCode>

### Nesting icons in Buttons

SVG elements can be used inside other elements and styled with the `sx` prop as well.

<LiveCode containerSx={`sx={{display: 'flex', justifyContent: 'space-around' }}`}>{`<Button>
  <svg sx={{ height: '14px', width: '14px', fill: 'white' }} viewBox="0 0 20 20">
    <path d="M13 8V2H7v6H2l8 8 8-8h-5zM0 18h20v2H0v-2z"/>
  </svg>
  Download
</Button>`}</LiveCode>
